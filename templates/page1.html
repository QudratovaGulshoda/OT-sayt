Windows operatsion tizimi


Windows Microsoft tomonidan shaxsiy kompyuterlar va serverlar uchun yaratilgan operatsion tizimlar oilasining nomidir.

Windows 10 — shaxsiy kompyuterlar va ish stantsiyalari uchun Microsoft tomonidan Windows NT oilasi tomonidan ishlab chiqilgan operatsion tizim. Windows 8.1`dan so`ng, tizim 9 ni chetlab o`tib, 10 raqamni oldi, Windows 10`ning server hamkori Windows Server 2016 hisoblanadi. Tizim turli xil qurilmalar, masalan, shaxsiy kompyuterlar, planshetlar, smartfonlar, Xbox One pristavkalari va boshqalar uchun birlashtirilgan bo`lishi uchun ishlab chiqilgan bo`lib, unda barcha qo`llab-quvvatlanadigan qurilmalar bilan birlashtirilgan rivojlanish platformasi va universal dasturlarning yagona do`koni mavjud. Windows 10 xizmat sifatida qo`llab-quvvatlash sikli davomida yangilanishlar bilan ta`minlanadi. Tizim chiqarilgandan keyingi birinchi yil davomida foydalanuvchilar Windows 7, Windows 8.1 va Windows Phone 8.1 ning litsenziyalangan nusxalarini ishlaydigan qurilmalarda Windows 10`ga bepul o`tishlari mumkin edi. Muhim yangiliklar qatoriga “Cortana” ovozli yordamchisi, bir nechta ish stoli yaratish va almashtirish qobiliyati va boshqalar kiradi. Windows 10 — bu Windows`ning so`nggi “qutidagi” versiyasi, barcha keyingi versiyalar faqat raqamli shaklda tarqatiladi. Windows 10 foydalanuvchi shartnomasi Microsoft`ga foydalanuvchi, uning Internetdagi faoliyati tarixi, kirish nuqtalari uchun parollar, klaviaturada yozilgan ma`lumotlar va boshqa ko`p narsalar haqida ko`p ma`lumot to`plash imkonini beradi. W3Schools statistik ma`lumotlariga ko`ra, Windows 10 Internetga ulanish uchun ishlatiladigan operatsion tizimlar orasida dunyoda birinchi o`rinni egallab turibdi.
